# 📄 Assistente Inteligente de Documentos com LangChain (RAG)

Projeto desenvolvido em Python com foco em aprendizado prático de LangChain, RAG (Retrieval-Augmented Generation) e busca semântica aplicada a documentos PDF.

## 🚀 Objetivo
Transformar documentos PDF em uma base consultável por linguagem natural.

## 🛠️ Tecnologias
- Python
- LangChain
- Hugging Face
- FAISS
- PyPDF

## ⚙️ Pipeline
1. Upload do PDF
2. Leitura
3. Chunking
4. Embeddings
5. Indexação
6. Recuperação

## 🔍 Exemplos de perguntas

- Qual é o período de inscrição?
- Liste os documentos exigidos para contratação conforme o edital
- Quais são os requisitos básicos para contratação temporária?
- Em quais situações o candidato pode ser eliminado do processo seletivo?

## 📈 Aprendizados
- RAG na prática
- Embeddings semânticos
- Busca vetorial
- Limitações de LLMs leves
